## Food Allergies

- Peanuts, peas (all other legumes: lentils, green beans, etc)
- Tree Nuts<sup>1</sup>
- Potatoes (and sweet potatoes)
- Mustard (in wasabi)
- Raw tomatoes
- Beans<sup>2</sup>
- Shellfish<sup>3</sup> (crab, lobster, clams, oysters)
- Bell peppers, paprika
- Zucchini
- Squash

## Food that is OK

1. Almonds
2. Soy
3. Shrimp
