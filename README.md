## Food Allergies

- Peanuts, peas (all other legumes: lentils, green beans, etc)
- Tree Nuts<sup>1</sup>
- Potatoes, sweet potatoes<sup>2</sup>
- Mustard
- Wasabi
- Raw tomatoes
- Beans<sup>3</sup>
- Shellfish<sup>4</sup> (crab, lobster, clams, oysters)
- Bell peppers, paprika
- Zucchini
- Squash

## Food that is OK

1. Almonds, coconut
2. Japchae
3. Soy
4. Shrimp
